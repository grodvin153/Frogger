2 - Control de movimiento

Actualización del juego

Para actualizar el movimiento del juego, vamos a definir la función reservada update.

Dentro de esta función llamaremos a las funciones update_player y update_vehicles que realizaremos en los siguientes pasos.

ver código

def update():
    update_player()
    update_vehicles()

Actualización del jugador

Para controlar el movimiento del jugador, vamos a definir una función con el nombre update_player.

  • Dentro de la función:
    • Definimos la velocidad del jugador, por ejemplo 3.
    • Si pulsamos a la tecla flecha izquierda (keyboard.left) y no nos salimos de la pantalla por el lado izquierdo (x>0)
      • Decrementamos (ir a la izquierda) la posición horizontal del jugador según la variable de velocidad.
      • Cambiamos la imagen del jugador a "frog_left".
    • Si pulsamos a la tecla flecha derecha (keyboard.right) y no nos salimos de la pantalla por el lado derecho (x<width)
      • Incrementamos (ir a la derecha) la posición horizontal del jugador según la variable de velocidad.
      • Cambiamos la imagen del jugador a "frog_right".

ver código

def update_player():
    player_speed = 3
    if keyboard.left and player.x > 0:
        player.x = player.x - player_speed
        player.image = 'frog_left'
    if keyboard.right and player.x < WIDTH:
        player.x = player.x + player_speed
        player.image = 'frog_right'
  • Si pulsamos a la tecla flecha arriba (keyboard.up) y no nos salimos de la pantalla por el lado superior (y>0)
    • Decrementamos (ir hacia arriba) la posición vertical del jugador según la variable de velocidad.
    • Cambiamos la imagen del jugador a "frog_up".
  • Si pulsamos a la tecla flecha abajo (keyboard.down) y no nos salimos de la pantalla por el lado derecho (y<height)
    • Incrementamos (ir hacia abajo) la posición vertical del jugador según la variable de velocidad.
    • Cambiamos la imagen del jugador a "frog_down".

ver código

def update_player():
    player_speed = 3
    if keyboard.left and player.x > 0:
        player.x = player.x - player_speed
        player.image = 'frog_left'
    if keyboard.right and player.x < WIDTH:
        player.x = player.x + player_speed
        player.image = 'frog_right'
    if keyboard.up and player.y > 0:
        player.y = player.y - player_speed
        player.image = 'frog_up'
    if keyboard.down and player.y < HEIGHT:
        player.y = player.y + player_speed
        player.image = 'frog_down'

Añadimos también la comprobación si hemos alcanzado la zona segura (posición vertical menor que 50).

De ser así, volvemos a colocar el jugador en la casilla de salida.

ver código

def update_player():
    player_speed = 3
    if keyboard.left and player.x > 0:
        player.x = player.x - player_speed
        player.image = 'frog_left'
    if keyboard.right and player.x < WIDTH:
        player.x = player.x + player_speed
        player.image = 'frog_right'
    if keyboard.up and player.y > 0:
        player.y = player.y - player_speed
        player.image = 'frog_up'
    if keyboard.down and player.y < HEIGHT:
        player.y = player.y + player_speed
        player.image = 'frog_down'
    if player.y < 50:
        player.pos = (WIDTH // 2, HEIGHT - 50)

Actualización de los vehículos

Para controlar el movimiento de los vehículos, vamos a definir una función con el nombre update_vehicles.

  • Dentro de la función:
    • Definimos la velocidad común de los vehículos, por ejemplo 3.
    • Para cada vehículo en la lista de vehículos
      • se incrementa su posición horizontal añadiendo el valor de la velocidad. Este incremento significa que se mueve hacia la derecha.
      • Si el vehículo toca el margen derecho (los vehículos se mueven hacia la derecha), es decir la posición x>ancho (width).
        • Se posiciona el vehículo por fuera del margen izquierdo (posición x = -50)

ver código

def update_vehicles():
    vehicle_speed = 3
    for vehicle in vehicles:
        vehicle.x = vehicle.x + vehicle_speed
        if vehicle.x > WIDTH:
            vehicle.x = -50

Vamos a repetir el mismo procedimiento para comprobar si el jugador ha colisionado con algún vehículo. Para ello

  • Dentro de la misma función update_vehicles:
    • Para cada vehículo en la lista de vehículos
      • Si el jugador (player) ha colisionado (collider) con el vehículo
        • el jugador se vuelve a posicionar en la casilla de salida.

ver código

def update_vehicles():
    vehicle_speed = 3
    for vehicle in vehicles:
        vehicle.x = vehicle.x + vehicle_speed
        if vehicle.x > WIDTH:
            vehicle.x = -50
    for vehicle in vehicles:
        if player.colliderect(vehicle):
            player.pos = (WIDTH // 2, HEIGHT - 50)
