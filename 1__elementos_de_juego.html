1 - Elementos de juego

Inicialización del juego

En primer lugar vamos a inicializar el juego:

  • importando la librería pygame zero pgzrun y la librería random.
  • añadiendo un título al juego, por ejemplo Frogger.
  • añadiendo las dimensiones, 600 de alto (height) y 800 de ancho (width).

ver código

import pgzrun
import random

TITLE = 'Frogger'
WIDTH = 800
HEIGHT = 600

Añadimos los elementos principales del juego

En primer lugar creamos el actor rana, utilizando la imagen "frog_up".

La posición horizontal será el centro (width/2) de la pantalla, y la posición vertical a 50 del borde inferior (height-50).

ver código

player = Actor("frog_up", (WIDTH // 2, HEIGHT - 50))

Para crear el conjunto de coches necesitaremos utilizar listas.

  • En primer lugar creamos una lista vacía.
  • A continuación indicamos el número de coches que vamos a crear, por ejemplo 8.
  • Y definimos la posición vertical inicial del primer coche, por ejemplo 50.
  • Vamos a repetir esta operación tantas veces como coches queremos añadir a la lista
    • Obtenemos la posición horizontal aleatoria del coche. Será un valor entre 0 y el tamaño de pantalla (width).
    • Incrementamos la posición vertical del coche en 40. De esta forma, cada coche será posicionado en una altura diferente (90, 130, 170...)
    • Se crea un coche en la posición horizontal y vertical definida anteriormente, con la imagen "car".
    • Añadimos el coche creado a la lista.

ver código

vehicles = []
y = 50
num_vehicles = 8
for i in range(num_vehicles):
    x = random.randint(0, WIDTH)
    y = y + 40
    car = Actor("car", (x, y))
    vehicles.append(car)

Dibujamos los elementos en pantalla

Para mostrar los elementos de juego por pantalla utilizaremos la función draw.

Dentro de esta función:

  • Rellenamos (fill) el fondo de la pantalla de color negro (black).
  • Dibujamos un rectángulo verde (green) desde la esquina superior izquierda (posición 0,0) de altura 50 y de ancho toda la pantalla (width).

ver código

def draw():
    screen.fill("black")
    screen.draw.filled_rect(Rect((0, 0), (WIDTH, 50)), "green")

A continuación añadimos los elementos de juego:

  • Añadimos el objeto player.
  • Añadimos cada uno de los vehículos que están dentro de la lista vehículos.

ver código

Dentro de la función draw

def draw():
    screen.fill("black")
    screen.draw.filled_rect(Rect((0, 0), (WIDTH, 50)), "green")

también añadimos el siguiente código:

    player.draw()
    for vehicle in vehicles:
        vehicle.draw()

Ejecución del juego

La última línea del código debe ser la ejecución del programa (go) a través de la librería pgzrun.

ver código

pgzrun.go()
